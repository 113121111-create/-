# **💻 電腦零組件電商網站 \- 後端核心系統 (PC Parts E-commerce Backend)**

這是一個專為電腦零件（如 CPU、主機板、顯示卡、機殼）設計的 RESTful API 後端專案。本專案的核心亮點在於利用 **PostgreSQL 的 JSONB 欄位**，完美解決了不同硬體擁有完全不同規格（例如：CPU 有腳位、顯卡有長度、機殼有限高）的技術痛點，並內建了 **DIY 組裝相容性檢查機制**。

## **🌟 核心特色**

* **動態規格儲存 (Dynamic Schema)：** 採用 PostgreSQL GIN 索引與 JSONB 技術，不需頻繁修改資料表結構即可自由上架各種類型的硬體。  
* **智慧相容性檢查 (Smart Compatibility Checks)：**  
  * 根據主機板腳位 (Socket) 自動過濾相容的 CPU。  
  * 自動比對顯示卡長度與機殼空間，防止買家發生「機構衝突（裝不下）」的窘境。  
* **高效能設計：** 針對動態規格欄位建立索引，確保在海量商品中依然能維持毫秒級的篩選速度。

## **🛠️ 開發技術棧 (Tech Stack)**

* **執行環境：** Node.js  
* **網頁框架：** Express.js  
* **資料庫：** PostgreSQL 12+  
* **依賴套件：** pg (PostgreSQL Client), dotenv (環境變數管理)

## **🚀 快速開始與安裝步驟**

### **1\. 複製專案與安裝依賴**

將專案下載至本地端，並在根目錄執行安裝指令：

npm install

### **2\. 資料庫建置 (PostgreSQL)**

1. 開啟你的 PostgreSQL 管理工具（如 pgAdmin 或 DBeaver）。  
2. 建立一個名為 pc\_store 的資料庫。  
3. 複製並執行專案中的 init.sql 檔案，這會自動建立資料表並寫入測試用的零件資料（Seed Data）。

### **3\. 設定環境變數 (.env)**

在專案根目錄建立一個 .env 檔案，並填入你的資料庫連線資訊：

PORT=3000  
DB\_HOST=localhost  
DB\_PORT=5432  
DB\_NAME=pc\_store  
DB\_USER=你的資料庫帳號 (預設通常是 postgres)  
DB\_PASSWORD=你的資料庫密碼

### **4\. 啟動伺服器**

執行以下指令啟動本地開發伺服器：

npm start

當看到 🚀 電腦零件後端系統已啟動：http://localhost:3000 與 PostgreSQL 資料庫連線成功！ 即代表運行成功。

## **🗺️ API 路由與使用說明 (API Documentation)**

### **1\. 商品管理 API**

#### **📦 取得商品列表**

* **網址：** GET /api/products  
* **查詢參數 (選填)：** category\_id (過濾特定分類)  
* **範例 URL：** http://localhost:3000/api/products?category\_id=1  
* **說明：** 分類 ID 對照：1: CPU, 2: 主機板, 3: 顯示卡, 4: 機殼。

#### **➕ 上架新硬體零件**

* **網址：** POST /api/products  
* **請求主體 (JSON Body 範例)：**  
  {  
    "category\_id": 3,  
    "name": "ASUS ROG RTX 4080 SUPER",  
    "brand": "ASUS",  
    "price": 36490,  
    "stock": 5,  
    "specifications": {  
      "length\_mm": 357,  
      "recommended\_psu\_watts": 750  
    }  
  }

### **2\. DIY 估價單相容性檢查 API**

#### **🔍 依主機板腳位撈出相容的 CPU**

* **網址：** GET /api/compatibility/cpus  
* **必要查詢參數：** socket (如：LGA1700, AM5)  
* **範例 URL：** http://localhost:3000/api/compatibility/cpus?socket=AM5  
* **回傳：** 系統會精準撈出所有屬於 AM5 腳位的 AMD 處理器，自動剔除 Intel 或舊款 AMD 產品。

#### **📏 檢查顯示卡是否裝得進機殼 (機構檢查)**

* **網址：** GET /api/compatibility/check-gpu-case  
* **必要查詢參數：** case\_id (機殼商品ID), gpu\_id (顯卡商品ID)  
* **範例 URL (安全放得下)：** http://localhost:3000/api/compatibility/check-gpu-case?case\_id=7\&gpu\_id=5  
* **範例 URL (太長裝不下)：** http://localhost:3000/api/compatibility/check-gpu-case?case\_id=8\&gpu\_id=6  
* **JSON 回傳範例 (衝突時)：**  
  {  
    "compatible": false,  
    "case\_max\_length\_mm": 300,  
    "gpu\_length\_mm": 336,  
    "message": "❌ 警報！顯示卡太長 (336mm)，該機殼最大只支援 300mm！"  
  }
